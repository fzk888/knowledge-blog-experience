`uv` 是一个速度很快的 Python 包管理工具，可以用来创建虚拟环境、安装依赖和运行项目。下面是一个常用项目初始化流程。

## 安装 uv

macOS：

```bash
brew install uv
uv --version
```

其他平台可以参考官方安装方式。

## 进入项目目录

```bash
cd /path/to/your/project
```

## 创建虚拟环境

默认 Python：

```bash
uv venv
```

指定 Python 版本：

```bash
uv venv --python 3.11
```

当项目依赖对 Python 版本敏感时，建议显式指定版本。

## 激活虚拟环境

macOS 或 Linux：

```bash
source .venv/bin/activate
```

Windows PowerShell：

```powershell
.venv\Scripts\Activate.ps1
```

激活成功后，终端前缀通常会显示虚拟环境名称。

## 安装依赖

从 `requirements.txt` 安装：

```bash
uv pip install -r requirements.txt
```

安装单个包：

```bash
uv pip install fastapi
```

查看已安装包：

```bash
uv pip list
```

## 运行项目

如果已经激活虚拟环境，可以直接运行：

```bash
python app/main.py
```

也可以使用：

```bash
uv run python app/main.py
```

## 退出虚拟环境

```bash
deactivate
```

## 常用命令速查

| 功能 | 命令 |
| --- | --- |
| 创建虚拟环境 | `uv venv` |
| 指定 Python 版本创建虚拟环境 | `uv venv --python 3.11` |
| 激活虚拟环境（macOS/Linux） | `source .venv/bin/activate` |
| 安装单个包 | `uv pip install fastapi` |
| 安装依赖文件 | `uv pip install -r requirements.txt` |
| 查看已安装包 | `uv pip list` |
| 删除虚拟环境 | `rm -rf .venv` |
